# 客户搜索工具

主动寻找有贷款需求的潜在客户，提高获客效率。

## 功能特性

- 🔍 **多平台搜索** - 百度、知乎、抖音、小红书
- 🎯 **智能意向分析** - AI判断需求强度，自动分类
- 📊 **线索管理** - 状态跟踪、优先级排序
- 💬 **触达模板** - 私信、评论、短信话术
- 📄 **报告生成** - 自动生成Markdown报告

## 快速开始

```bash
# 进入目录
cd /home/codespace/clawd/customer-monitor

# 搜索关键词
echo "贷款" | python3 customer_search.py

# 或直接指定
python3 customer_search.py "征信逾期"
```

## 使用方法

### 基本搜索

```bash
# 交互式输入
python3 customer_search.py

# 指定关键词
python3 customer_search.py "贷款利息"
```

### 自定义配置

编辑 `data/config.json`:

```json
{
  "keywords": ["贷款", "征信", "急需资金"],
  "exclude_words": ["诈骗", "骗子"],
  "sources": ["百度搜索", "知乎", "抖音搜索", "小红书"],
  "intent_threshold": 60,
  "max_results": 50
}
```

## 输出文件

| 文件 | 说明 |
|------|------|
| `leads_report_*.md` | 线索报告 |
| `leads_*.json` | 结构化数据 |
| `outreach_*.json` | 外呼数据 |

## 意向等级

| 等级 | 评分 | 说明 |
|------|------|------|
| 🟢 高 | 70-100 | 急需贷款、征信问题 |
| 🟡 中 | 40-69 | 了解阶段、比较价格 |
| 🔴 低 | 10-39 | 潜在需求 |
| ⚪ 无 | 0-9 | 排除词命中 |

## 触达模板

触达模板保存在 `templates/templates.json`，可根据需求修改：

```json
{
  "私信": {
    "高意向": "您好，看到您在咨询贷款问题...",
    "中意向": "您好，看到您在了解贷款相关..."
  }
}
```
