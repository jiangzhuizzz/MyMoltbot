# 客户搜索与触达系统

主动寻找有贷款需求的潜在客户，并自动化触达转化。

## 功能特性

### 1. 客户搜索 (customer_search.py)

- 🔍 **多平台搜索** - 百度、知乎、抖音、小红书
- 🎯 **智能意向分析** - AI判断需求强度，自动分类
- 📊 **线索管理** - 状态跟踪、优先级排序

### 2. 自动化触达 (auto_reach.py)

- 🤖 **自动发送** - 私信、评论、短信
- ⏰ **定时发送** - 模拟延迟，避免频率限制
- 📈 **效果追踪** - 回复率、转化率统计
- 🛡️ **防封机制** - 频率控制、变体话术

## 快速开始

```bash
# 进入目录
cd /home/codespace/clawd/customer-monitor

# 1. 搜索客户
python3 customer_search.py "贷款"

# 2. 自动触达
python3 auto_reach.py
```

## 触达系统详解

### 平台配置 (data/platform_config.json)

| 平台 | 每日上限 | 发送间隔 | 触达方式 |
|------|----------|----------|----------|
| 抖音 | 50条 | 30-120秒 | 私信+评论 |
| 小红书 | 30条 | 60-180秒 | 私信+评论 |
| 知乎 | 20条 | 2-5分钟 | 私信+评论 |
| 百度 | 10条 | 5-10分钟 | 评论 |
| 短信 | 100条 | 5-10秒 | - |

### 触达状态

| 状态 | 说明 |
|------|------|
| ⏳ 待触达 | 等待发送 |
| 📤 发送中 | 正在发送 |
| ✅ 已发送 | 发送成功 |
| 💬 已回复 | 客户回复 |
| 🎉 已转化 | 客户转化 |
| ❌ 发送失败 | 发送失败 |
| 🔴 被封禁 | 账号被封 |

### 优化建议

**提升回复率**
1. 个性化话术 - 根据客户问题定制
2. 最佳时段 - 分析回复率最高的时间
3. 跟进机制 - 发送后2-3天电话跟进

**避免封禁**
1. 控制频率 - 不超过平台限制
2. 内容变化 - 使用变体话术
3. 账号矩阵 - 多账号分散风险
