# CRM客户关系管理系统

管理客户全生命周期，从线索到成交。

## 功能特性

### 1. 客户管理
- ✅ 添加/编辑/删除客户
- ✅ 客户信息完整记录（姓名、电话、来源、意向等）
- ✅ 客户标签管理
- ✅ 高级搜索和筛选

### 2. 跟进管理
- ✅ 跟进记录（电话、微信、面谈等）
- ✅ 跟进提醒
- ✅ 待跟进客户列表

### 3. 成交管理
- ✅ 成交记录
- ✅ 佣金统计
- ✅ 转化率分析

### 4. 数据分析
- ✅ 客户统计（总数、状态分布、意向分布）
- ✅ 销售漏斗
- ✅ 来源分析
- ✅ 自动生成报告

## 快速开始

```bash
# 进入目录
cd /home/codespace/clawd/crm-system

# 查看帮助
python3 crm_cli.py help

# 列出所有客户
python3 crm_cli.py list

# 添加客户
python3 crm_cli.py add 张三 13800138000 抖音

# 搜索客户
python3 crm_cli.py search 贷款

# 添加跟进
python3 crm_cli.py follow <客户ID> 电话 客户有意向

# 查看统计
python3 crm_cli.py stats

# 生成报告
python3 crm_cli.py report
```

## 客户状态

| 状态 | 说明 |
|------|------|
| 新客户 | 刚录入的客户 |
| 已联系 | 首次联系完成 |
| 跟进中 | 持续跟进 |
| 已筛选 | 需求已明确 |
| 方案沟通 | 提供方案中 |
| 谈判中 | 细节谈判中 |
| 成交 | 成功成交 |
| 流失 | 客户流失 |

## 意向等级

| 等级 | 说明 |
|------|------|
| 高意向 | 明确需求，预算充足 |
| 中意向 | 有需求，考虑中 |
| 低意向 | 潜在需求，不紧急 |
| 无意向 | 无贷款需求 |

## 数据文件

| 文件 | 说明 |
|------|------|
| `data/customers.json` | 客户数据 |
| `data/followups.json` | 跟进记录 |
| `data/deals.json` | 成交记录 |
| `data/*.csv` | 导出数据 |

## 与客户搜索集成

```bash
# 从搜索结果导入客户
python3 crm_cli.py import ../customer-monitor/data/leads_贷款_*.json

# 生成完整报告
python3 crm_cli.py report
```

## 销售漏斗

```
新客户      →  已联系  →  跟进中  →  已筛选  →  方案沟通  →  谈判中  →  成交
   100%        80%        60%        40%         30%          20%       10%
```
