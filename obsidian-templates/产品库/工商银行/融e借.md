## 申请条件

### 必要条件
- [ ] 公积金连续缴纳满6个月
- [ ] 有稳定打卡工资
- [ ] 征信良好（无逾期、无不良记录）
- [ ] 年龄22-60岁

### 加分项（满足更容易通过）
- [ ] 工商银行代发工资
- [ ] 工商银行存量客户
- [ ] 学历本科以上
- [ ] 有房产

### 禁止条件（这些情况不能申请）
- [ ] 当前有逾期
- [ ] 近2年连3累6
- [ ] 征信查询过多（3个月>6次）
- [ ] 已退休人员

## 申请要求

1. 身份证原件
2. 公积金账号密码
3. 实名制手机号
4. 征信授权

1. 扫码申请（工行APP或微信小程序）
2. 在线审批（人脸识别+资料填写）
3. 审批通过（1-3个工作日）
4. 签约放款（最快当天到账）

## 优势

1. 利率低（3.65%起）
2. 审批快（1-3天）
3. 额度高（最高30万）
4. 无抵押无担保
5. 随借随还

## 劣势

1. 对征信要求严格
2. 公积金要求较高
3. 额度因人而异
4. 部分客户可能降额

## 常见问题

**Q: 这个产品适合什么样的人？**
A: 适合公积金缴纳基数高（>8000）、征信良好、在优质单位工作的客户。

**Q: 审批需要多长时间？**
A: 一般1-3个工作日，最快当天审批通过。

**Q: 如果被拒了怎么办？**
A: 建议3个月后再次申请，期间保持良好征信记录。
